## 🛠️ Quick Start:

### Step 1: Install Python
- Download Python 3.8+ from https://python.org
- Check "Add Python to PATH" during installation

### Step 2: Extract Files
- Extract this zip file to a folder (e.g., `C:\ScriptForge\`)

### Step 3: Install Dependencies
```cmd
cd C:\ScriptForge
pip install streamlit requests python-docx python-dotenv
```

### Step 4: Get an API Key (FREE)
1. Go to https://platform.deepseek.com/
2. Sign up for free account
3. Create API key in "API Keys" section
4. Copy your key

### Step 5: Run ScriptForge
```cmd
streamlit run app.py
```
Then open: http://localhost:8501

### Step 6: Enter API Key
- Paste your DeepSeek API key in the sidebar
- Start generating scripts!

## 🎬 Your First Script:
